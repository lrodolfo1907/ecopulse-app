# ecopulse-app

## Quick Start

To get started with the ecopulse-app, please follow these instructions:

1. Clone the repository:
   ```sh
   git clone https://github.com/lrodolfo1907/ecopulse-app.git
   cd ecopulse-app
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the application:
   ```sh
   npm start
   ```

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Docker, Heroku

## Features

- User authentication and authorization
- Real-time data processing
- Interactive dashboard
- RESTful API endpoints
- Responsive design for mobile devices

## Limitations

- Currently, the application does not support offline mode.
- Limited user roles available; additional roles are planned for future releases.
